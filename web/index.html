Install

From download to a working Chrome extension

1

Download the zip

Click Install Extension to download textpull-ai-extension.zip to your computer.

2

Extract the folder

Unzip the download so you have a normal folder named extension with manifest.json inside it.

3

Open Chrome extensions

Go to chrome://extensions in Chrome and turn on Developer mode in the top-right corner.

4

Load unpacked

Click Load unpacked and choose the extracted extension folder, not the zip file itself.

5

Pin and use it

Pin TextPull AI from the Chrome toolbar, open any article or doc, and launch the extension popup to start testing.

Quick note

Chrome does not install unpacked extensions directly from a website click. The download gives users the source package, then Chrome loads it from the extracted folder.
